App

(PHP 7, UI 0.9.9)

導入

The App object represents a libui context, providing methods to enter into, and quit, the main UI loop.

クラス概要

UI\App {
/* メソッド */
protected integer onShouldQuit ( array $windows )
protected onTick ( void )
public quit ( void )
public run ([ int $flags ] )
}
